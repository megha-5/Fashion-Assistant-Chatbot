# Conversational-Fashion-Assistant-Spring-Boot-Chatbot-
Maeve is an AI-powered fashion chatbot designed to deliver a personalized and elegant shopping experience. Built with Spring Boot on the backend and Botpress Webchat on the frontend, Maeve acts as a virtual fashion assistant that engages users in real-time, answers queries, and records their conversations.
